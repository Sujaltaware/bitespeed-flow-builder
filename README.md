# BiteSpeed Chatbot Flow Builder

A React-based chatbot flow builder using [React Flow](https://reactflow.dev/).

## Live Demo

> Add your Vercel/Netlify link here after deployment

## Features

- **Text Nodes** — Drag & drop message nodes onto the canvas
- **Nodes Panel** — Extensible panel; add new node types by editing `NodesPanel.jsx`
- **Edges** — Connect nodes; source handle allows only ONE outgoing edge; target handles allow multiple
- **Settings Panel** — Click any node to edit its text via the settings panel
- **Save Validation** — Shows error if more than one node has no incoming connections

## Setup & Run

```bash
npm install
npm run dev
```
